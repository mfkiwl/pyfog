Signal Processing¶

This package provides support for Allan deviation and other algorithms.

pyfog.signal_processing.allan_deviation(data, rate)[source]¶

Returns the Allan deviation. Makes use of oadev from the allantools repository

Parameters:
  • data (array_like(float)) – The data to be processed
  • rate (float) – The sampling rate in Hz
Returns:

  • tau (list of float) – The taus used in the Allan deviation
  • dev (list of float) – The Allan deviations.

>>>foo()
